Problem Statement: Simple Student Report Card

Your task is to create a basic Java program that represents a student's report card. The goal is to learn how to create a **class**, instantiate it to create an **object**, and then use that object to perform a simple task.

Functional Requirements

  • Student Information: The program needs to store a student's name and their score for three subjects: Math, Science, and English.
  • Calculate Grade: The program should calculate the average score for the three subjects and determine if the student passed or failed. A student passes if their average score is 50 or greater.
  • Display Result: The program must display the student's name, their average score, and whether they passed or failed.

Technical Requirements

  • Create a Student Class: Design a single class named Student.
  • Fields: The class should have public fields to store the student's name and their scores. For example: String name, int mathScore, int scienceScore, and int englishScore.
  • Methods: Implement the following methods within the Student class:
    • A method to calculate the average score.
    • A method to check if the student passed or failed.
    • A method to display all the information (name, average score, pass/fail status).
  • Demonstrate in Main: In the main method, create a **single object** of the Student class.
    • Set the student's name and their three subject scores directly.
    • Call the method to display the final report card.

Testing Requirements

Your main method must demonstrate the program works by testing the following scenarios:

  1. Passing Student: Create a student object with scores that should result in a passing grade (e.g., Math: 80, Science: 75, English: 90). The output should correctly show a passing status.
  2. Failing Student: Create a second student object with scores that should result in a failing grade (e.g., Math: 40, Science: 35, English: 45). The output should correctly show a failing status.
