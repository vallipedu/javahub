Problem Statement: Simple Loan Eligibility

Your task is to create a very simple Java program to check if a person is eligible for a bank loan. This problem focuses on the most basic programming concepts: creating a **class** and using **conditional logic** to make a decision.

Functional Requirements

  • Applicant Data: The system needs to check an applicant based on their annual income and credit score.
  • Eligibility Rule: An applicant is eligible for a loan if **both** of the following conditions are true:
    • Annual income is **at least $30,000**.
    • Credit score is **at least 650**.
  • Output: The program must simply print "Eligible" or "Not Eligible".

Technical Requirements

  • Create a LoanApplicant Class: Design a single class named LoanApplicant.
  • Fields: Use **private** fields for annualIncome and creditScore.
  • Methods:
    • A constructor to initialize the applicant's income and credit score.
    • public boolean isEligible(): A method that checks both conditions and returns a `true` or `false` value.
  • Demonstrate in Main: In your `main` method, create two `LoanApplicant` objects and test the two required scenarios.

Testing Requirements

Your main method must demonstrate the program works by testing the following scenarios:

  1. Eligible Applicant: Create an applicant who meets both criteria (e.g., income: $45,000, credit score: 700). The program should print "Eligible".
  2. Ineligible Applicant: Create an applicant who fails at least one criterion (e.g., income: $25,000, credit score: 680). The program should print "Not Eligible".
