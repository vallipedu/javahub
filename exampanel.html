Instructions for the Lab Exam

Dear Students,

Welcome to the Java Lab Exam. Your success in this exam depends on your ability to not only write code but also to think like a professional software developer. Follow these steps for your assigned problem:

  1. Find Your Assignment: Locate your assigned problem statement below. Click on the corresponding link to view the details.
  2. Understand the Requirements: Read the problem statement carefully. Pay close attention to the **Technical Requirements**, **Functional Requirements**, and **Testing Requirements**.
  3. Analyze & Design: Before you start coding, spend time analyzing the problem. Think about the classes, interfaces, and methods you'll need. Sketch out a simple class diagram or a design plan.
  4. Implement: Write clean, well-commented, and efficient Java code. Adhere to best practices and coding standards.
  5. Test: Use the provided testing requirements to verify that your solution works as expected. A well-tested solution is a key part of a successful project.
