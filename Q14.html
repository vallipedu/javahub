Problem Statement: Boxes with Inheritance

Your task is to create a set of Java classes to model different types of boxes: a generic **Box**, a specialized **Cube**, and a **Weighted Box**. This problem focuses on applying **inheritance** and **encapsulation** to build a class hierarchy.

Functional Requirements

  • Box: Must have dimensions (width, height, depth) and a method to calculate its volume.
  • Cube: A special type of box where all dimensions are equal. Its constructor should handle this.
  • WeightBox: A type of box that has an additional property: weight. It must also have a method to return its total weight.

Technical Requirements

  • 1. Create a Box Class:
    • This will be the parent class. It should have **private** fields for width, height, and depth.
    • Include a constructor to initialize these dimensions.
    • Provide a public method `getVolume()` that returns the calculated volume.
  • 2. Create a Cube Class:
    • This class must **extend** the `Box` class.
    • Its constructor should take a single argument for the side length and pass this value to the parent `Box` class constructor for all three dimensions.
  • 3. Create a WeightBox Class:
    • This class must also **extend** the `Box` class.
    • It should have an additional **private** field for weight.
    • Its constructor should accept four arguments: width, height, depth, and weight. It must pass the dimensions to the parent `Box` class constructor.
    • Add a public method `getWeight()` to return the weight.
  • 4. Demonstrate Polymorphism:
    • In your `main` method, create a List of type Box.
    • Add objects of each class (`Box`, `Cube`, and `WeightBox`) to this list.
    • Loop through the list and print the volume of each object. Notice how Java handles the correct calculation for each type.

Testing Requirements

Your main method must demonstrate the program works by testing the following scenarios:

  1. Individual Object Tests:
    • Create a `Box` object with dimensions 2x3x4. Print its volume (24).
    • Create a `Cube` object with a side length of 5. Print its volume (125).
    • Create a `WeightBox` object with dimensions 2x3x4 and a weight of 10. Print its volume (24) and weight (10).
  2. Polymorphism Test:
    • Create a List and add one instance of each class (`Box`, `Cube`, and `WeightBox`) to it.
    • Write a `for` loop to iterate through the list. Inside the loop, print the volume of each box. The output should show the correct volume for each object, even though they are all treated as `Box` objects in the list.
